Home Server


Overview:

GitHub - GitHub Link to Project
- Built a home lab in microsoft azure to simulate an enterprise network and vilnerable machines.
- Deployed a domain controller on Windows Server 2022 along with two Windows Server 2019 machines, utilizing Azure Virtual Machines. Deployed two Windows 11 VMs and joined to the domain.
- Created multiple VNETS within the Resource Group to simulate multiple network subnets.
- Utilized Azure Monitor to monitor performance and create alerts.
-
- Server Monitoring: Uptime Kuma Was Used To Ensure That My Server Is Healhy And Running As Intended
                      • Alerts - To Notify Me When Services Go Down
                      • HTTP Status Code / TCP Monitoring / Ping Monitoring / DNS Monitoring

Securing my Home Server:

Router Security:

- Switching From Generic Router to a More Secure Alternative: Pfsense Router
- Subnets: Using Two Subnets:
                      • Subnet 1: Gaming Systems, Phones, Guess Devices, Plex
                      • Subnet 2: Personal Computer, Firewall Access, Admin For Devices
 - Routing All Network Traffic through VPN
- Firewall To Prevent Unauthorized Access Into and Out of My Network

System Security:

- Overview of How i Secured My Home Server:
- Ubuntu Operating System: Provides Better Security To Alternatives (Windows)
- Watchtower: Automatically Updates My Docker Images

Data Backup:

Fault Tolerance, The Measures i Undertook To Ensure The Safety of My Data:

- Maintaining 3 Copies Of My Data via the 3-2-1 Rule:
- 3 Copies of My Data: Primary (SSD), 1st Backup (HDD), 2nd Backup (Cloud):
- My 1st Copy being my Primary Device my SSD, Which is Mirrored to my Backup HDD.
- This HDD is synchronized with my Google Cloud.
